# Fluent numbers

Tired of plain numbers in you code?  
Ever dreamt of writing several lines of code instead of few digits?  

### How does it work?

The boring way :'(
```
long value = 1964;
```

The exciting way!
```
long value = Numbers.positive().one().nine().six().four().create();
```

### Requirements

Java 8 or higher

### The future

- Basic arithmetic operations
- And more!
